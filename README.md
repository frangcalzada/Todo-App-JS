1. Clone repository
2. Run ```npm install``` to rebuild the node modules
3. Run the devServer ```npm run dev```

## Production
1. Run ```npm run build```
2. Take the ```dist``` folder and deploy it

## In this project, I use:
1. HTML
2. CSS
3. Vanilla JS
4. Store concept
5. LocalStorage
6. UUID
